
Hướng dẫn upload website:
1. Vào https://www.netlify.com/ hoặc https://vercel.com/ và đăng ký tài khoản miễn phí.
2. Bấm "New site" -> "Import an existing project" -> Upload toàn bộ thư mục này (index.html + style.css).
3. Sau khi up thành công, bạn sẽ có 1 link web ngay lập tức.

Chúc bạn thành công!
